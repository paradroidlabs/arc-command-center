# Context Tracker Roadmap

This document outlines upcoming features, UX/UI improvements, and planned explanations that will be embedded into the app interface.

## Planned Features
- **In-App Tooltips**: Add contextual explanations to metrics like Survival Rate, Net Profit, and Time Topside.
- **Raid Data Export**: Allow exporting raw JSON data from the UI.
- **Custom Challenge Tracker**: Allow creators to define custom challenges and natively track specific metrics, game items, or events to measure their progress.

## UX / UI Improvements
- Refine the map performance table sorting.
- Add hover effects to the Recent Raids list to show deeper details.

## Known Issues
- Currently tracking none.
